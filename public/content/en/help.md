# How to Use Storm Kamishibai

## 1. Select an image

- Choose the image file you want to display.
- Adjust zoom if needed.

## 2. Tune color conversion

- Adjust color order and draw flags.
- Re-run conversion after changing options.

## 3. Tune Lua settings

- Configure card size, offsets, and skips.
- Fine-tune based on your target screen.

## 4. Generate output

- Run conversion to generate Lua code.
- Download as ZIP when needed.
